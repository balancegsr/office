# Office Skill 系列

> **FormCraft** series — 把粗糙输入赋予专业形式，一句话变成可交付的格式化产物

一组办公场景 Skill，通过对话驱动，将你的文字、数据、笔记转化为可直接使用的专业格式化输出。

每个 Skill 专注一种输出格式。你提供内容，Skill 负责结构化、排版和交互——生成的产物浏览器直接打开，零依赖。

## 设计原则

**Skill 管格式，用户管内容。**

每个 Skill 都是格式转换工具，不替你做内容判断。你给什么素材，它就用最优的方式呈现——像一个只管排版不改稿的设计师。

这意味着：

- 非专业人士也能产出专业格式的交付物（Skill 保证呈现质量）
- 专业用户同样适用（你提供专业内容，Skill 负责视觉表达）

## 共享设计

系列内所有 Skill 遵循统一的设计约束：

- **单文件交付**：每个产物都是一个自包含文件，不依赖外部服务
- **对话驱动**：用户不需要了解 HTML/CSS/JS，只管提供内容和偏好
- **技术透明**：所有技术选择（图表库、动画方案、布局算法）由 Agent 自主判断，不暴露给用户
- **输出以 HTML 为主**：浏览器直接打开，跨平台零安装
